### 2. DeepMind AI Control Roadmap
- **What:** Google DeepMind published its **[AI Control Roadmap](https://storage.googleapis.com/deepmind-media/DeepMind.com/Blog/securing-the-future-of-ai-agents/gdm-ai-control-roadmap.pdf)** (June 18) — a defense-in-depth framework for securing AI agents deployed inside Google infrastructure. Key elements: treats agents as "insider threats" using MITRE ATT&CK; D1-D4 detection levels based on evasion capability; R1-R3 prevention/response based on harm potential; analyzed **1 million coding agent trajectories** to inform live monitoring. Accompanying paper: ["Three Layers of Agent Security"](https://storage.googleapis.com/deepmind-media/DeepMind.com/Blog/securing-the-future-of-ai-agents/three-layers-of-agent-security.pdf).
- **Why it matters:** This is the most concrete industry attempt to operationalize "alignment plus" — system-level security that works even when alignment is imperfect. The 1M trajectory analysis is a significant dataset for the agent safety community. The framework is explicitly designed to be reusable by other labs and policymakers.
- **Sources:**
  - [DeepMind Blog](https://deepmind.google/blog/securing-the-future-of-ai-agents/)
  - [The Verge](https://www.theverge.com/tech/952899/google-deepmind-announced-an-ai-control-roadmap-for-improving-ai-agent-security)
