# LLM Research Digest — 2026-05-22

## BOTTOM LINE (READ THIS FIRST)

Google I/O dominated the day with **Gemini 3.5 Flash GA** (83.6% MCP Atlas, strongest agentic model; $1.50/$9/MTok) and **Antigravity 2.0** — a standalone desktop agent IDE that replaces Gemini CLI (shutting down June 18). Anthropic's "Code with Claude" London event shipped **self-hosted sandboxes** for Managed Agents, **MCP tunnels**, and announced the **June 15 Agent SDK credit split** that breaks programmatic Claude usage from subscriptions. ## NEW THIS WEEK - RANKED BY KARL-RELEVANCE

### 1. Google Gemini 3.5 Flash — GA (Google I/O May 19) #BREAKTHROUGH
- **Karl-relevance:** HIGH — This is the most capable agentic model released this week; directly relevant to local-first AI infra and agent orchestration
- **BS indicator:** LOW — Benchmarks verified from DeepMind's official page with methodology links; benchmark numbers checked against third-party analysis
- **What:** First model in the Gemini 3.5 family, GA May 21. Designed for frontier performance across agents and coding at speed.
- **Standout fact:** **83.6% MCP Atlas — strongest agentic model** (beats GPT-5.5's 75.3%, Claude Sonnet 4.6's 69.5%)
- Benchmark highlights: Terminal-Bench 2.1 Agentic terminal coding: **76.2%** (vs Gemini 3 Flash 58.0%); GDPval-AA Elo: **1656** (vs GPT-5.5's 1769); SWE-Bench Pro: **55.1%**; UI Control OSWorld: **78.4%**; CharXiv Reasoning: **84.2%**
- Pricing: **$1.50/MTok input, $9/MTok output** (1M context window); 4x faster than comparable models
- Model ID: `gemini-3.5-flash`; available via Gemini API, AI Studio, Antigravity, Enterprise Agent Platform
- Also announced: **Gemini Omni Flash** (video generation), **Gemini 3.5 Pro** (coming June)
- *Source URLs:* [DeepMind Gemini page](https://deepmind.google/models/gemini/) | [Google Cloud blog](https://cloud.google.com/blog/products/ai-machine-learning/innovations-from-google-io-26-on-google-cloud) | [Artificial Analysis benchmark](https://www.labellerr.com/blog/google-gemini-3-5-flash/) | [TokenMix pricing](https://tokenmix.ai/blog/gemini-3-5-pro-release-date-google-io-2026)
- *Why this matters:* Sets the new baseline for agentic coding. The 83.6% MCP Atlas score is the highest multi-step agent benchmark published to date.

### 3. Anthropic Self-Hosted Sandboxes + MCP Tunnels — Managed Agents #BREAKTHROUGH
- **Karl-relevance:** HIGH — Self-hosted sandboxes directly relevant to any enterprise/regulated Claude Agent deployment; MCP tunnels enable private network access for agents
- **BS indicator:** LOW — Primary source (Anthropic announcement) + independent analysis from Fortune and devtoolpicks
- **What:** Two new features for Claude Managed Agents announced at Code with Claude London (May 19):
  - **Self-hosted sandboxes** (public beta): Tool execution moves into customer-controlled infrastructure (Cloudflare microVMs, Modal GPU, Vercel VMs, Daytona, or any container). Agent loop still runs on Anthropic's servers.
  - **MCP tunnels** (research preview): Secure outbound-only tunnel from private network to Anthropic routing — no public endpoints needed for internal MCP servers.
- Supported providers at launch: Cloudflare, Modal, Vercel, Daytona, self-hosted
- Limitations: Agent loop stays on Anthropic; orchestration metadata still flows through Anthropic; not available on Claude Platform on AWS; Memory not supported yet
- *Source URLs:* [Anthropic announcement via Fortune](https://fortune.com/2026/05/21/claude-code-london-anthropic-ai-software-engineering/) | [devtoolpicks deep analysis](https://devtoolpicks.com/blog/anthropic-self-hosted-claude-agents-mcp-tunnels-indie-hackers-2026)
- *Why this matters:* Breaks the "all Claude agent data must leave your network" assumption. For compliance-heavy use cases, this is a real blocker removal.

### 4. Anthropic June 15 Agent SDK Credit Split — Pricing Shockwave #SIGNAL
- **Karl-relevance:** HIGH — Directly impacts programmatic Claude usage in OpenClaw and any automation pipeline
- **BS indicator:** LOW — Primary source from devtoolpicks with detailed pricing math; Hacker News community reaction corroborates
- **What:** Starting June 15, 2026, programmatic usage of Claude via subscription plans moves to a separate monthly credit pool.
  - Hits: Agent SDK usage, `claude -p` command, Claude Code GitHub Actions, third-party apps (OpenClaw, Conductor, Zed, Jean)
  - Unaffected: Interactive Claude Code in terminal/IDE, web/desktop/mobile chat, Claude Cowork
  - Credit amount matches subscription price: Pro $20/mo = $20 Agent SDK credit; Max 5x = $100; Max 20x = $200
  - At Sonnet 4.6 API rates ($3/M input, $15/M output), $20 covers ~6.6M input or 1.3M output tokens
  - Exceed credit → either extra usage billing at full API rates OR requests stop until refresh
- Context: Third subscription change in six weeks (April 4 third-party agent ban → April 21 Claude Code removal test from Pro → May 15 credit split)
- *Source URLs:* [devtoolpicks analysis](https://devtoolpicks.com/blog/anthropic-splits-claude-subscriptions-agent-sdk-credit-june-2026) | [Anthropic @ClaudeDevs announcement](https://twitter.com/ClaudeDevs)
- *Why this matters:* If OpenClaw uses Claude via Agent SDK or programmatic calls, June 15 changes the economics. Heavy automation users may need to move to direct API keys on Developer Platform.

### 5. Cursor Composer 2.5 — Open-Weight Base at 1/10th Price #SIGNAL
- **Karl-relevance:** HIGH — Open-weight base model (Kimi K2.5) with aggressive pricing; relevant for local-first AI infra and cost analysis
- **BS indicator:** MED — Multiple sources confirm benchmarks, but some claims are Cursor-published; third-party validation from Artificial Analysis
- **What:** Released May 18, 2026. Built on Moonshot AI's Kimi K2.5 (1T total params, 32B active, MoE). RL fine-tuned on real-time user data from millions of Cursor sessions.
- Benchmarks: SWE-Bench Multilingual **79.8%**, CursorBench v3.1 **63.2%** — matches Opus 4.7 on benchmarks
- Pricing: **$0.50/MTok input, $2.50/MTok output** (Standard tier); Fast tier $3/$15
- Comparison: Opus 4.7 is $15/$75; GPT-5.5 same band. A 2M-token agentic session costs **$2.20 on Composer 2.5 vs tens of dollars** on competitors.
- Base model license: Modified MIT (attribution required for services >100M monthly users or $20M/month revenue)
- *Source URLs:* [Artificial Analysis](https://artificialanalysis.ai/articles/cursor-composer-2-5-coding-agent-index) | [The New Stack](https://thenewstack.io/cursor-composer-benchmarks/) | [dev.to community report](https://dev.to/om_shree_0709/cursor-just-released-composer-25-heres-what-actually-changed-for-ai-coding-agents-51fc)
- *Why this matters:* Demonstrates that open-weight models + RL fine-tuning on proprietary data can compete with frontier labs at a fraction of cost. Kimi K2.5 is becoming strategic infrastructure for Western AI products.

### 7. Hirundo Security-Hardened Gemma 4 — Weight-Level Unlearning #SIGNAL

- Featured in Google DeepMind's official Gemmaverse showcase (May 21)
- *Source URLs:* [VentureBeat](https://venturebeat.com/business/google-deepmind-features-hirundos-security-hardened-gemma-4-model-outperforms-llms-170x-its-size-on-security) | [Business Wire press release](https://www.businesswire.com/news/home/20260521997520/en/) | [HuggingFace model card](https://huggingface.co/hirundo-io/gemma-4-E4B-it-reduced-prompt-injection)
