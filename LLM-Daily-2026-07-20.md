**Date:** July 17, 2026 (published)
**Signal:** `#SIGNAL`
**Evidence Tier:** T2 | **Confidence:** 80%
**Sources:**
- [OpenAI: A Scorecard for the AI Age](https://openai.com/index/a-scorecard-for-the-ai-age/)

**OpenAI published a new framework paper** proposing "Useful Intelligence per Dollar" as the ultimate metric for evaluating AI investment. The paper introduces four evaluation dimensions and uses GPT-5.6 (Sol/Terra/Luna tiers) as the primary case study. Key findings:

- **DeepSWE v1.1 benchmark**: GPT-5.6 Sol reaches 72.7% on long-horizon engineering tasks, above Claude Fable 5's 69.9%, at 36.2% lower estimated API cost (54% fewer output tokens).
- **Tiered model economics**: Sol = flagship reasoning; Terra = performance/cost balance; Luna = fastest/cheapest. The framework argues the right tier depends on the task, not a single "best" model.
- **Four dimensions of value**: (1) Work accomplished — how much useful output per dollar; (2) Cost per successful task — accounting for retries and human review; (3) Dependability — ready-to-use vs. needs correction vs. needs escalation rates; (4) Scaling returns — does each AI dollar buy more work as usage grows?
- **ChatGPT Work positioning**: Tied to ChatGPT Enterprise security/compliance, enabling deeper workflow integration with oversight.

**Independent check:** Read the OpenAI blog post fully. The DeepSWE benchmark comparison (72.7% vs 69.9%) is specific and verifiable. The framework is clearly positioned as a marketing piece for GPT-5.6's tiered pricing, but the underlying numbers (cost per successful task, retry rates) are concrete claims worth tracking.

**Why it matters:** This is OpenAI's first public attempt to frame AI value in terms of "work accomplished per dollar" rather than raw benchmark scores. For Karl: if he's evaluating GPT-5.6 vs. alternatives for agentic workflows, the DeepSWE v1.1 numbers (72.7% with 36.2% lower API cost) are worth testing directly. The framework's four dimensions could become a useful evaluation lens for his own model selection.
