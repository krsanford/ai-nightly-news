### F01 — Z.ai GLM-5.2: 744B MoE, 1M Token Context, MIT License
**Date:** June 13, 2026 (API release: June 16, 2026)
**Sources:** [MarkTechPost](https://www.marktechpost.com/2026/06/14/z-ai-launches-glm-5-2-with-a-usable-1m-token-context-two-thinking-effort-levels-and-no-benchmarks-at-launch/) · [Gigazine](https://gigazine.net/gsc_news/en/20260615-z-ai-glm-5-2/) · [AwesomeAgents](https://awesomeagents.ai/models/glm-5-2/) · [DEV.to](https://dev.to/max_quimby/run-glm-52-locally-the-open-model-nobody-can-ban-pnb) · [OpenRouter](https://openrouter.ai/z-ai/glm-5.2)
**Domain:** Model release / open-weight

Z.ai released GLM-5.2, the third major iteration in its GLM-5 line. Key specs:
- **744B total parameters, 40B active per token** (MoE architecture)
- **1,000,000-token context window** (glm-5.2[1m]) — up from GLM-5.1's 200K
- **131,072 max output tokens**
- **MIT license**, open weights on the way (API release today)
- Two thinking-effort levels: High and Max
- First-day support for Claude Code, Cline, OpenCode, Roo Code, Goose

**Context:** Released three days after the US government ordered Anthropic's Fable 5 / Mythos 5 offline. Z.ai founder Jie Tang explicitly framed it as a response: "The sudden restriction of certain frontier models is deeply regrettable... we are even more convinced that science should be global." The community reaction on HN was notably positive about Chinese labs being open with their work.

**Benchmarks:** None published at launch. GLM-5.1 scored 58.4 on SWE-bench Pro and 1530 Elo on Code Arena — competitive at the time. Z.ai is presumably still running evaluations.

**Significance:** This is a direct insurance play against export-control disruptions. Open-weight, MIT-licensed, with 1M context that puts it in a class of its own for long-horizon agentic coding. API goes live today (June 16).

### F03 — Anthropic Credit Overhaul: End of Unlimited Subscription Subsidy
**Date:** Effective June 15, 2026 (announced May 14)
**Sources:** [DigitalApplied](https://www.digitalapplied.com/blog/anthropic-claude-credit-overhaul-june-15-2026) · [Ina Fried/Axios via DigitalApplied]
**Domain:** Pricing / API

Anthropic separates Agent SDK and `claude -p` (headless) usage from Pro, Max, Team, and Enterprise subscription pools. A new monthly dollar credit — sized to match each plan's subscription fee — replaces the prior "unlimited" subsidy. Credits refresh monthly with no rollover. Enterprise Standard seats get $0 in Agent SDK credit; only Enterprise usage-based seats ($20) and Premium seats ($200) include it.

**The math:** A Pro subscriber paying $20/month could previously run up to $300–600 in API-equivalent compute (15–30× subsidy). This ends that structural subsidy. Agent SDK requests now flow at standard API rates when monthly credit runs out — or simply fail if no usage credits are enabled.

**Significance:** "All-you-can-eat" AI subscriptions may not survive the agent era. For teams running production automation on Claude subscriptions, this is the most significant billing change since Claude Code launched.
