# LLM Research Digest — May 28, 2026

## BOTTOM LINE (READ THIS FIRST)

Anthropic released **Claude Opus 4.8** today (May 28) at the same price as 4.7 ($5/$25/M), but with meaningful benchmark gains across coding (SWE-Bench Pro: 69.2%, up from 64.3%), reasoning (USAMO 2026: 96.7%, up from 69.3% — a 27-point jump), and honesty (misalignment incidence dropped from 2.5 to ~1.9). Fast mode is now 3× cheaper at 2.5× speed. Claude Code also launched **Dynamic Workflows** (research preview) — plan, dispatch hundreds of parallel subagents, verify each other's findings. **Claude Mythos** will roll out to all customers "in the coming weeks." Separately, Anthropic announced a **$65B fundraise at $965B valuation**, surpassing OpenAI as the world's most valuable AI startup. OpenAI had no new model release today (GPT-5.5 remains current).

## TREND WATCH

1. **Anthropic's cadence is accelerating.** Opus 4.6 (Feb 5) → 4.7 (Apr 16) → 4.8 (May 28). Roughly 6-7 weeks between releases, faster than their usual cadence. This tracks with public pressure from OpenAI and the Mythos rollout timeline.
2. **Honesty as a benchmark dimension is maturing.** Anthropic's "4x less likely to let code flaws pass" framing — backed by misalignment incidence dropping from 2.5 to ~1.9 — represents a new class of evaluation. If other labs adopt similar metrics, the coding model landscape could shift on dimensions beyond raw accuracy.
3. **The AI startup valuation race just flipped.** Anthropic at $965B surpassing OpenAI at $852B is a symbolic but potentially material inflection. The $65B raise (vs OpenAI's March round) gives Anthropic serious compute firepower for the Mythos rollout and Claude scaling.
4. **Dynamic workflows represent a new agentic paradigm.** Claude Code's parallel subagent approach — plan, dispatch hundreds of independent agents, verify each other's findings — is a structural shift from sequential agent loops. If this generalizes across coding tools, it changes how large-scale codebase tasks are scoped.
5. **Local inference continues its steady maturation.** Ollama v0.30 (May 27) with MLX preview support on Apple Silicon; Qwen3.5 35B available; the "default quantization" shift is making larger models accessible on consumer hardware. No breakthrough today, but the trend line holds.

### Claude Opus 4.8 — Open-Source Relevance Assessment
- **Not open-weight.** Opus 4.8 is a closed model available only via API (Claude API, Bedrock, Vertex AI, Foundry). No local deployment path.
- **What it means for local-first infra:** The benchmark gains (SWE-Bench Pro: 69.2%, USAMO: 96.7%) raise the bar for what local models need to match. The closest open-weight competitors are Llama 4 (released early 2025) and Qwen 3.6 — both significant gaps from Opus 4.8's numbers.
- **Why this matters for Karl:** If local-first is the strategy, the gap between Opus 4.8 and the best locally deployable models is widening. This puts pressure on the local inference stack to either close the gap (through techniques like speculative decoding, which EAGLE 3.1 addresses) or accept a hybrid architecture where frontier models handle the hardest tasks.
