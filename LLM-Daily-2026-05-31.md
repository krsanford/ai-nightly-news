# AI Research Digest — 2026-05-31

## BOTTOM LINE (READ THIS FIRST)

The week of May 28–31 is dominated by **Anthropic's Claude Opus 4.8 launch** (May 28) — the most cost-effective frontier model for agentic workloads, with 69.2% on SWE-bench Pro, adaptive thinking, and fast mode at $10/$50/M tokens. But the real story is what's coming: **Claude Mythos-class models go public "in the coming weeks"** — a cybersecurity-tier model scaled to all customers. On the infra side, **Anthropic self-hosted sandboxes for Claude Managed Agents hit AWS** (May 29) with MCP tunnels, and **OpenAI published its Frontier Governance Framework** (May 28) mapping safety practices to CA and EU law. Meanwhile, **GPT-5.6 leaks point to a June launch** with rumored 1.5M context windows. For Karl: Opus 4.8 fast mode is worth A/B testing against GPT-5.5 for agent skills; the Mythos release will force a strategic decision on what tier your OpenClaw agents need.

### 1. Claude Opus 4.8 — System Card + Full Benchmarks (Anthropic, May 28)
**#BREAKTHROUGH** — **The most cost-effective frontier model for agentic workloads** · finding ID `F05`
- SWE-bench Pro: **69.2%** (+4.9 pts from 4.7), SWE-bench Verified: **88.6%**
- GDPval Elo: **1890** (+121 over GPT-5.5), USAMO 2026: **96.7%** (up from 69.3%)
- Super-Agent: only model to complete every case end-to-end at parity cost vs GPT-5.5
- Online-Mind2Web (computer use): **84%** — best browser-agent score tested
- Legal Agent Benchmark: first model to break 10% all-pass
- **Fast mode: $10/$50/M tokens** (3× cheaper than 4.7), 2.5× faster
- **Adaptive thinking**: triggers reasoning only when needed, reducing wasted tokens
- Defaults to **high effort**; "extra" and "max" tiers available
- Four times less likely than 4.7 to let code flaws pass unremarked
- Alignment: misaligned behavior rates "substantially lower" than 4.7, similar to Mythos Preview
- Karl-relevance: **HIGH** — directly impacts agent skill cost/quality tradeoffs; fast mode worth A/B testing against GPT-5.5
- Evidence + Confidence: `T2 → T3 · ~90% (multi-source + independent gateway verification by TrueFoundry)`
> [!tip] Action — A/B test Opus 4.8 fast vs GPT-5.5 on the same agentic tasks in OpenClaw
> The cost delta (~$1.45 vs ~$1.66 per SWE-bench task at p50 latency) is material at scale.
