# LLM Research Digest — May 26, 2026

## BOTTOM LINE (READ THIS FIRST)

Anthropic is in the middle of a historic week: it's closing a **$30B+ funding round at $900B+ valuation** (surpassing OpenAI for the first time), simultaneously preparing Claude Mythos 1 for public release via Claude Code and Security, and announcing a $200M Gates Foundation partnership. ### Microsoft Kills Claude Code Internally by June 30 #SIGNAL
- **Karl-relevance:** HIGH — directly relevant to agentic coding economics; proves token-based pricing breaks at enterprise scale
- **Evidence:** T2 — The Verge (original scoop), Windows Central, NextWeb, StartupFortune, KuCoin all confirm Microsoft canceling Claude Code licenses for Experiences & Devices division
- **BS indicator:** LOW — multiple independent sources corroborate; the financial logic is clear and independently verifiable
- **What:** Microsoft is canceling most internal Claude Code licenses for its Experiences & Devices group (Windows, M365, Outlook, Teams, Surface), redirecting engineers to GitHub Copilot CLI by June 30. The official reason: toolchain unification. The unofficial: token billing burned through the annual AI budget in months. At Uber, Claude Code usage jumped from 32% to 84% of engineers; individual engineers spent $500–$2,000/month on tokens.
- **Standout fact:** **This is the first major company to publicly acknowledge that agentic coding tools are too expensive at scale** — Microsoft, the world's largest software company, proved token-based pricing breaks when usage becomes heavy and persistent. "I'm back to the drawing board" — Uber's CTO.
- **Why this matters for Karl:** If even Microsoft can't sustain Claude Code costs at enterprise scale, the unit economics of agentic coding are fundamentally unsettled. This validates the thesis that cheaper models (Claude Code's 50% limit increase was a defensive maneuver; OpenAI's Codex consumes 4.2x fewer tokens) will win the economics war.
- Source URLs: [NextWeb](https://thenextweb.com/news/microsoft-claude-code-retreat-ai-cost) [StartupFortune](https://startupfortune.com/microsofts-claude-pullback-shows-ai-coding-still-has-a-budget-problem/) [KuCoin](https://www.kucoin.com/news/flash/microsoft-discontinues-internal-use-of-claude-code-due-to-high-costs)
