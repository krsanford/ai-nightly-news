**Key validations:**
- **Artificial Analysis Intelligence Index v4.1 = 51** — tops the index ahead of MiniMax-M3 (44), DeepSeek V4 Pro (44), Kimi K2.6 (43). Beats GPT-5.5 on several long-horizon coding benchmarks at ~1/6 the price.

- **Architecture:** MLA + DSA from prior GLM/DeepSeek lineage, plus new IndexShare (reuses sparse-attention top-k indices across layer groups to reduce 1M-token inference cost).
- **SWE-bench Pro: 62.1%** — beats GPT-5.5 and Claude on this benchmark.
- **Available free** via HuggingFace Inference Providers for a limited window; also on Ollama.

**Why it matters:** The consensus from multiple independent practitioners is that GLM-5.2 is the first open-weight model that genuinely feels "frontier-adjacent" in daily use — not just benchmaxxed. If Z.ai can maintain this trajectory, an "Open Fable-class" model (without Anthropic's export-control risk) could emerge within months.
