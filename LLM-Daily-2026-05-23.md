### 1. OpenAI Codex v26.519 "Codex Thursday" — Goal Mode GA, Appshots, Locked Mac #BREAKTHROUGH

- Browser use improvements: advanced annotation mode, faster asset extraction, read-only JS context, tab grouping, Chrome extension cleanup, reliability fixes.
- ChatGPT Business adds richer Codex workflows with appshots, goal mode GA, browser annotations, locked computer use for eligible Macs, shared plugins for workspace reuse.

### 3. NVIDIA Verified Agent Skills — Enterprise Agent Security Framework #SIGNAL
- **Karl-relevance:** MEDIUM-HIGH — Relevant to agent security infrastructure; could be relevant if Karl builds or deploys multi-agent systems with tool use
- **BS indicator:** LOW — Official NVIDIA developer blog published; framework details include scanning, signing, skill cards, SkillSpector risk checks for prompt injection/tool poisoning/hidden instructions. GitHub repo exists (NVIDIA/NemoClaw).
- **What:** NVIDIA launched Verified Agent Skills on May 22, a capability governance framework using three layers: (1) Scanning — detects hidden instructions, prompt injection, trigger abuse, excessive agency, tool poisoning; (2) Signing — cryptographically verifies skill origin and integrity; (3) Skill Cards — metadata about declared purpose vs. actual behavior.
- Technical detail: SkillSpector checks agent-specific risks beyond file-level analysis — whether a skill's declared purpose matches its requested access level and bundled behavior.
- Ecosystem: Part of NVIDIA Agent Toolkit; early preview NemoClaw repo exists on GitHub (NVIDIA/NemoClaw) for running agents in NVIDIA OpenShell runtime with managed inference.
- *Source URLs:* [NVIDIA Technical Blog](https://developer.nvidia.com/blog/nvidia-verified-agent-skills-provide-capability-governance-for-ai-agents/) | [GitHub NemoClaw](https://github.com/NVIDIA/NemoClaw) | [Metaverse Post coverage](https://mpost.io/nvidia-expands-ai-governance-strategy-with-verified-skills-risk-scanning-and-trust-metadata-for-agents/)
- *Why this matters:* As agents gain more autonomous capability (like Codex's new locked-Mac feature), security governance becomes critical. NVIDIA is positioning itself as the trust layer for agent ecosystems — relevant if Karl deploys agents with tool access.

## TREND WATCH

1. **The "background agent" inflection point is here.** OpenAI's locked-Mac Codex + Anthropic's Dreaming memory consolidation + Google's Gemini Spark (24/7 agent that runs on dedicated VMs) all point to the same conclusion: agents are becoming background processes, not foreground tools. The competitive frontier has shifted from "who can write better code" to "who can run longer without supervision."

2. **Compute economics are the real bottleneck.** Anthropic's $45B/3yr SpaceX deal ($15B/yr) is a floor price signal for frontier-scale capacity. xAI already sold ALL Colossus I to Anthropic and is seeking more customers. Meta raised 2026 AI capex guidance to $125-145B. The compute arms race is creating a oligopoly of infrastructure providers (AWS, GCP, Azure, xAI/Colossus).

3. **Agent security frameworks are emerging as a category.** NVIDIA's Verified Agent Skills (May 22) adds scanning/signing/skill-cards to the agent governance toolkit. This follows the EU AI Act enforcement deadline (August 2026) and growing regulatory pressure on autonomous agent deployments.

4. **Talent war intensifies.** Karpathy joining Anthropic's pre-training team (May 19) is the highest-profile movement between labs in 2026. Combined with Anthropic's Milan office expansion and ongoing hiring from OpenAI/Google, the talent concentration is accelerating.

5. **Anthropic vs OpenAI competitive dynamics are explicit.** May 13 saw two mirror-image moves: Anthropic raised Claude Code limits (defensive, for individual devs) while OpenAI offered free Codex months to enterprise switchers (offensive, for IT decision-makers). This "bottom-up vs top-down" pattern is now the dominant competitive strategy in agent tools.
